# Nate's Production Pro v1.0 — Java 17 New Repo Version

This is the clean new-repository version.

## Upload these files/folders to GitHub

```text
assets/nates_logo.png
lib/main.dart
pubspec.yaml
README.md
.github/workflows/build_apk.yml
```

If `.github` will not upload because it is hidden, manually create this file in GitHub:

```text
.github/workflows/build_apk.yml
```

Then paste the workflow from this project.

## Build

Go to:

Actions → Build Nate's Production Pro APK → Run workflow

Download the artifact:

```text
Nates-Production-Pro-v1-APK
```
